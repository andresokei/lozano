Sociogram.app is an innovative platform built with Laravel, aimed at providing educators and social researchers with the tools to create and analyze sociograms effectively. This web application facilitates social network analysis, allowing users to visualize interpersonal relationships within groups, such as classrooms or research samples.

Key features of Sociogram.app include an intuitive dashboard for managing groups and students, as well as a range of data visualization options that help users uncover social dynamics. The platform also integrates advanced analytics, offering insights that are valuable for understanding group interactions, peer relationships, and social hierarchies.
